## Working note

PWSH Studio is a developer workspace inspired by PowerShell: quick commands, route-aware output, project notes, and contact actions all live in one focused interface.

## System architecture

The system uses typed data, App Router pages, MDX engineering notes, a contact API route, lightweight motion, and client components only where interaction is needed.
