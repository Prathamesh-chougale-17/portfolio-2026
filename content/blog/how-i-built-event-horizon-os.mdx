## Mission premise

Event Horizon OS is a portfolio disguised as a spacecraft operating system. The goal is to make skills, projects, writing, and contact feel like connected missions rather than isolated cards.

## System architecture

The system uses typed data, App Router pages, MDX research logs, a contact API route, lightweight WebGL atmosphere, and client components only where interaction is needed.
